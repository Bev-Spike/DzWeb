# 一眼丁真Web项目
## 项目简介
本项目的的初衷是作为C++Linux网络编程的入门练手项目，同时结合自己的理解，配合网络热门的丁真表情包，实现一个用C++编写而成的Web服务器，向互联网提供检索、上传丁真表情包的服务。
- 使用线程池 + 非阻塞socket + epoll实现的IO复用 + 事件处理（模拟Proactor） 的并发模型。
- 使用状态机解析HTTP请求报文，支持解析GET请求
- 实现异步日志系统，记录服务器运行状态
